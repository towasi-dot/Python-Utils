# Python-Utils
A couple of functions that are must-haves in Python.

# Clear
Just clear the entire console by using this function (optimazed for Windows, Linux, Mac, and this PyCharm terminal).

Just type 
```
from utils.Clear import clear
```
and then 
```
clear()
```
to use it

# Math Functions
It's exactly what you think, a couple of helpfull mathematical functions:
1. Factorial (to use run factorial(n)) - guess you know what it is
2. T(n) - you may not know this but it is like factorial but with adding instead of multiplying
3. Root (to use run root(n, type), type is 2 by default) - it's just a root
4. Average (to use run average(numbers), numbers should be a list of intigers) - it just counts the arythmetical mean of all numbers in list

Just type
```
from utils.MathFuncs import *
```
and then run the chosen function
